# My Portfolio Website

A clean, responsive personal portfolio website built with **HTML**, **CSS**, and **JavaScript**. It features a modern design with a dark/light theme toggle and smooth navigation between key sections like Home, Projects, Services, and Contact.

## 🔗 Live Link

**View Live Website:** https://shivfoolio.netlify.app

## 🚀 Deployment Tech

*   **Platform:** Netlify

## 🚀 Features

*   **Responsive Design:** Looks great on mobile, tablets, and desktops.
*   **Theme Toggle:** Switch between Light and Dark modes (persists on refresh).
*   **Sections:** Home, About, Services, Projects, Contact Us.
*   **Fast Performance:** Built with vanilla technologies (no heavy frameworks).

## 🛠️ Tech Stack

*   **HTML5:** Semantic markup and structure.
*   **CSS3:** Flexbox, Grid, CSS Variables, and Animations.
*   **JavaScript (ES6+):** DOM manipulation and theme logic.
